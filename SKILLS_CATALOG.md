# Skills Catalog

This catalog lists the agent skills available in this repository, imported from the reputable [Antigravity Kit](https://github.com/cnp762/antigravity-kit).

## Frontend & UI
- `react-patterns`: Advanced React patterns and performance optimization.
- `nextjs-best-practices`: Next.js App Router and server components.
- `tailwind-patterns`: Best practices for Tailwind CSS.
- `frontend-design`: Modern UI patterns and design principles.
- `ui-ux-pro-max`: Extensive collection of styles, palettes, and fonts.
- `mobile-design`: UI/UX patterns for mobile applications.

## Backend & API
- `api-patterns`: RESTful and GraphQL API design standards.
- `nestjs-expert`: NestJS architecture and modules.
- `nodejs-best-practices`: Node.js async handling and structure.
- `python-patterns`: Python coding standards and FastAPI.
- `database-design`: Schema design and normalization.
- `prisma-expert`: Prisma ORM usage and migrations.

## Infrastructure & DevOps
- `docker-expert`: Docker containerization and composition.
- `deployment-procedures`: CI/CD pipelines and deployment strategies.
- `server-management`: Linux server administration and maintenance.
- `bash-linux`: Essential Bash scripting and Linux commands.
- `powershell-windows`: PowerShell scripting for Windows environments.

## Quality Assurance & Testing
- `testing-patterns`: General testing strategies and frameworks.
- `webapp-testing`: End-to-end testing (e.g., Playwright).
- `tdd-workflow`: Test-Driven Development workflows.
- `systematic-debugging`: Structured approaches to debugging.
- `performance-profiling`: Analyzing and optimizing application performance.
- `code-review-checklist`: Standards for effective code reviews.
- `lint-and-validate`: Linter configurations and validation rules.

## Security
- `vulnerability-scanner`: Identifying common security vulnerabilities.
- `red-team-tactics`: Offensive security and penetration testing concepts.

## Planning & Architecture
- `architecture`: System architecture patterns and principles.
- `plan-writing`: Writing effective project plans and requirements.
- `brainstorming`: Techniques for idea generation and problem solving.
- `app-builder`: scaffolding complete applications.
- `mcp-builder`: Building Model Context Protocol servers.

## Specialized
- `game-development`: Game logic and mechanics.
- `seo-fundamentals`: Search Engine Optimization basic principles.
- `geo-fundamentals`: Geolocation and mapping fundamentals.
- `i18n-localization`: Internationalization and localization strategies.
- `clean-code`: General clean code principles (applies to all languages).
- `behavioral-modes`: Definitions of different agent personas/modes.
- `parallel-agents`: Patterns for multi-agent coordination.
- `documentation-templates`: Templates for various documentation types.
